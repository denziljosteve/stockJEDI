# stockJEDI User Guide

Welcome to stockJEDI! This platform provides institutional-grade investment analysis.

## Getting Started
1. **Search a Ticker:** Use the search bar on the landing page or navbar (e.g., `AAPL`, `TSLA`).
2. **Dashboard Overview:** 
   - View real-time pricing and the AI-generated **Recommendation** (Buy/Hold/Sell).
   - Review the **1-Week Prediction** probability bars to understand the ML model's directional bias.
3. **Deep Dive Analysis:** Read the Groq-generated institutional report covering Strengths, Weaknesses, Risks, and the core Investment Thesis.
4. **Portfolio Management:** Create an account to add stocks to your Watchlist or track virtual transactions in your Portfolio.

## Understanding the Confidence Score
- The platform outputs a "Confidence Score" (High/Medium/Low). This represents the alignment between the Technical indicators, Fundamental data, and ML prediction probabilities.
